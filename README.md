# Expense Tracker

> Single page application for tracking expenses.

---

### Setup & Installation

1.  **Dependencies**
    Enter both `backend` and `frontend` folders:
    ```bash
    npm install
    ```

2.  **Environment**
    Duplicate `.env.example` as `.env` in both folders and set your `PORT`.

3.  **Run**
    Access the **localhost** port where the frontend is running.

---

### Key Features

* **View** example expenses.
* **Create** new entries.
* **Update** existing expenses.

---

**Built with React, Next.js, NestJS & TypeScript**
